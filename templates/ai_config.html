Add one or more prompts. With OWLv2, each prompt is a comma-separated label list. With Florence-2 or Qwen, each prompt is an instruction describing anomalies/safety events to monitor in live video. Instruction-monitor modes raise alerts only when instructed anomalies are spotted.
